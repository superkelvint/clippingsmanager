    How To Use This Page

    This page helps you collect notes in a clean structure: Sections, optional Subsections, and Entries.

    Browser requirement: editing and file saving in this page only work in Chromium-based browsers (for example: Chrome, Edge, Brave, Arc, Opera).

    1. Start And Stop Editing

    • Click Enable Editing to make changes.
    • Click Exit Editing to lock the page again.
    • When locked, nothing can be edited by accident.

    2. Add Your Content

    • Use + Add Section for a new big topic.
    • Use + Add Entry inside a section for individual notes.
    • Use + Add Subsection inside a section when you want one more level of grouping.
    • Use + Add Entry inside a subsection for individual notes.
    • Use the red delete buttons in editing mode to remove sections, subsections, or entries.
    • New section/subsection titles auto-focus so you can type right away.

    3. Write Notes (Simple Formatting)

    • In note text boxes, press Ctrl/Cmd + B for bold.
    • Press Ctrl/Cmd + I for italic.
    • Press Enter for a new line.
    • For section and subsection titles, pressing Enter saves that title line (no extra line break).

    4. Table Of Contents (TOC)

    • The TOC updates as you edit.
    • Use Show Entries / Hide Entries to include note titles in the TOC.

    5. Saving

    • Saves happen automatically while editing.
    • You can force a save with Ctrl/Cmd + S.
    • A green “Saved” message confirms a successful save.

    6. Reordering

    • When editing is on, drag using the ⋮⋮ handles to reorder sections, subsections, or entries within the same container.

    7. Reset Everything (Destructive)

    • Click Reset Document to clear all sections/subsections/entries and reset the title.
    • You must type the exact page title to confirm.

    8. Create A New Document

    • In your file manager, copy this HTML file and rename the copy.
    • Open the copied file in your browser to start a clean new document.
