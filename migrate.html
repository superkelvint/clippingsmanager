Clippings Manager — Migrator

Upload an older clippings.html notes file and download a migrated copy based on the latest template.

This never uploads your file anywhere; it runs entirely in your browser.
If you are running this from file:// (not GitHub Pages), fetch may be blocked. In that case, uncheck the box and select a downloaded latest clippings.html.
Migration copies these fields from your old file into the new template: <title>, <h1 id="main-title">, <main id="app-root">, and <script id="highlight-palette-data">.
